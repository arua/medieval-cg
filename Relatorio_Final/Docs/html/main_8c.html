Detailed Description

Este eh o arquivo principal do programa do trabalho final de Computacao Grafica. Neste arquivo esta a funcao main que eh chamada quando o programa eh executado.

Funcao chamada pelo OpenGL quando uma tecla eh utilizada.

Quando uma tecla normal, ou seja, alfa-numerica eh utilizada, o OpenGL dispara essa funcao, que modifica a variavel global keystate que por sua vez sera utilizada pela funcao keyboardOp para realizar alguma alteracao no estado atual do jogo.
